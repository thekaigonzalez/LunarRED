ArgWizard

A simple, elegant, memory-safe library for argument parsing. Argument-Wizard serves as a drop in replacement for getopt, being as easy, if not easier to implement and setup than a majority of other argument parsing libraries. ArgWizard is 20-50 milliseconds faster than getopt.

How?

If you want to read up on how ArgWizard's internals work, we recommend you check out the
Interface document, that comes with ArgWizard's source code, as well as an updated copy which is available on this website.
